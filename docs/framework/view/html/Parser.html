InheritanceParser Object

This class is a combination of two things, a html cleaner that takes html strings and convertes it into xml and my html parser. The html parser takes that xml and creates / renders the html document. The cleaner part is based on the stuff below, but modified in many ways for wrapper http://www.buzzware.com.au/_blog/Coffee_into_Code/post/Pure_Actionscript_HTML_Parser_(with_thanks_to_John_Resig)/ 14th May 2009: The following was adapted to ActionScript 3.0 by Gary McGhee of Buzzware Solutions (email gary at buzzware . com . au) Which is based on this below HTML Parser By John Resig (ejohn.org) (of JQuery fame) Original code by Erik Arvidsson, Mozilla Public License http://erik.eae.net/simplehtmlparser/simplehtmlparser.js http://ejohn.org/files/htmlparser.js
