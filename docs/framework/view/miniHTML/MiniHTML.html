cdata()method
public function cdata(target:DisplayObjectContainer, xml:XML):Text

A text node or unknown tag is sent to this method to be rendered. A Text node is just a extened TextField, anything that can be set inside of a ActionScript TextField object can be defined in your CSS style sheet to modify this object. The width and height are base of of the parent.

Parameters

target:DisplayObjectContainer — The DisplayObjectContainer that the created Element will be added to.
 
xml:XML — The XML node that this Element will be rendered from

Returns
Text — Returns the DisplayObjectContainer that was created
div()method 
public function div(target:DisplayObjectContainer, xml:XML):DisplayObjectContainer

A div tag is sent to this method to be rendered. A basic Element is created and all of the attributes from the node are combined with the styles from the style sheet to style and then render this element. After the Element is added to the display list the element is positioned based on its styles. Only the most basic properties can be used within the style sheet to style these elements.

Parameters

target:DisplayObjectContainer — The DisplayObjectContainer that the created Element will be added to.
 
xml:XML — The XML node that this Element will be rendered from

Returns
DisplayObjectContainer — Returns the DisplayObjectContainer that was created
