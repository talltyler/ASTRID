Because of the vast differences in DisplayObjects in Flex as opposes to ActionScript only projects this class gives you the oppertunity to extend the base DisplayObject Class you would like all DisplayObjects in the framework to extend. By default we use Sprite because it has everything a project needs but If you want a Flex based project you can extend UIElement and you will be able to use Flex components within everything.
