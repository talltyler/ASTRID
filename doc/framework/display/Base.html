Classpublic class Base
InheritanceBase flash.display.Sprite
SubclassesBoot, RendererBase

If you are using display objects for anything, I would highly recomend usnig this as a base class. This class extends Sprite and manages any event listeners that are applied to it, has an init method for when it has been added to stage and cleans everything up when it is removed from the display list or when you call destroy();
